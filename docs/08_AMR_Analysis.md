# Antimicrobial Resistance (AMR) Analysis

## Objective

To identify known antimicrobial resistance genes in the genome of *Colwellia psychrerythraea* using ABRicate.

## Databases

- CARD
- NCBI AMR

## Results

| Database | Resistance genes detected |
|----------|--------------------------:|
| CARD | 0 |
| NCBI AMR | 0 |

## Interpretation

No known antimicrobial resistance genes were detected in the genome using the CARD and NCBI AMR databases. This suggests that the analyzed strain does not carry recognized resistance determinants represented in these databases.

## Conclusion

The genome showed no detectable antimicrobial resistance genes under the databases and search criteria used in this analysis.
